Consulta

HTML (Hypertext Markup Language)

Es el lenguaje estándar utilizado para crear y diseñar páginas web. Se compone de elementos y etiquetas que estructuran el contenido y le dan formato.

WWW (World Wide Web)

WWW, o World Wide Web, es un sistema de información global que se encuentra en Internet y que permite acceder a documentos de hipertexto. Estos documentos se pueden visualizar y enlazar utilizando un navegador web. El término "www" generalmente forma parte de las direcciones de sitios web

(Hypertext Transfer Protocol)

Es el protocolo utilizado para transferir datos en la web, permitiendo que los navegadores soliciten y reciban páginas web de servidores web.

TCP/IP (Transmission Control Protocol/Internet Protocol)

Es el conjunto de protocolos que permite la comunicación y la conexión de dispositivos en redes y en internet. TCP se encarga de la entrega ordenada y confiable de los datos, mientras que IP se encarga del enrutamiento y direccionamiento de los datos.

DNS (Domain Name System)

DNS es un sistema que traduce los nombres de dominio (por ejemplo, www.ejemplo.com) en direcciones IP numéricas que las computadoras pueden entender. Esto facilita la navegación web y la conexión a servidores.

CDN (Content Delivery Network)

Un CDN es una red de servidores distribuidos geográficamente que almacena copias de contenido web, como imágenes, archivos de estilo y scripts. El objetivo es acelerar la entrega de contenido al usuario final al servirlo desde el servidor más cercano.

FTP (File Transfer Protocol):

FTP es un protocolo utilizado para la transferencia de archivos entre un cliente y un servidor en una red. Permite cargar y descargar archivos desde un servidor web a una computadora local.
